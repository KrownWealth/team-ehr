# Frontend (WeCareEHR UI)

Built with:

- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui
- HeroUI
- Recharts

### Run Locally

```bash
cd frontend
npm install
npm run dev


App runs at http://localhost:3000
Features Added

Full landing page with hero + 4 feature cards
Responsive design
Global GeneralSans font
Reusable header & footer

```
